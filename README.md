# AI Circuit Mentor (Virtual Electronics University)

High-performance AI engineering learning platform for electronics and circuit education.

## Stack

- Frontend: Next.js App Router, React TypeScript, TailwindCSS, WebSocket, PWA, IndexedDB.
- Backend: FastAPI async APIs, Redis realtime layer, PostgreSQL, vector DB integration pattern.
- AI: LLM orchestrator, CV circuit scanner pipeline, speech tutor pipeline hooks, RL feedback loop worker.
- Deployment: Docker microservices with offline college mode + cloud SaaS mode.

## Quick start

```bash
cd infra/docker
docker compose up --build
```

- Frontend: http://localhost:3000
- Backend docs: http://localhost:8000/docs

## Key modules

- Notebook++: note explain + revision cards.
- Voice Tutor: low-latency streaming channel scaffold.
- Numerical Solver: step-by-step explanations.
- Circuit Scanner: image -> components -> netlist -> simulation stub.
- Virtual Lab: real-time websocket sync.
- Analytics: weak concept detection endpoint.
- Licensing: offline campus key verification.
- Billing: credit consumption + monetization plumbing.
